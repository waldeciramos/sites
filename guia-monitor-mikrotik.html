Monitor MikroTik
com ngrok

Guia completo de instalação, configuração e exposição pública do monitor de rede via ngrok.

01
Estrutura do Projeto

Crie a pasta do projeto e os arquivos necessários. O monitor roda em Python com Flask na porta 8080.

cmd / powershell 
# Criar pasta do projeto
mkdir C:\monitor_mikrotik
cd C:\monitor_mikrotik
Arquivos do projeto: app.py (servidor Flask), templates/index.html (dashboard), ngrok.exe (túnel público)
02
Instalar Dependências Python

Instale as bibliotecas necessárias para o monitor funcionar.

cmd / powershell 
pip install flask routeros-api
03
Rodar o Monitor Local

Inicie o servidor Flask. Ele ficará disponível em localhost:8080.

cmd / powershell 
cd C:\monitor_mikrotik
python app.py
Acesso local: http://localhost:8080
04
Instalar e Configurar o ngrok

Baixe o ngrok em ngrok.com/download, extraia o ngrok.exe dentro da pasta C:\monitor_mikrotik.

Crie uma conta gratuita em ngrok.com e autentique com seu token:

cmd / powershell 
ngrok config add-authtoken SEU_TOKEN_AQUI
Atenção: Substitua SEU_TOKEN_AQUI pelo token do seu painel em dashboard.ngrok.com/get-started/your-authtoken
05
Expor o Monitor Publicamente

Com o servidor Python rodando, abra outro terminal e execute:

cmd / powershell 
cd C:\monitor_mikrotik
ngrok http 8080

O resultado será parecido com:

Session Status         online
Account               waldeci (Plan: Free)
Version               3.39.6
Region                South America (sa)
Forwarding           https://skirmish-equate-marrow.ngrok-free.dev → http://localhost:8080
Importante: Mantenha este terminal aberto. Se fechar, o túnel cai. A URL muda a cada reinício no plano gratuito.
06
Comandos Úteis do ngrok
Comando O que faz
ngrok http 8080 Expõe a porta 8080 via HTTPS (uso principal)
ngrok http 80 Expõe a porta 80 (HTTP padrão)
ngrok tcp 22 Expõe SSH (porta 22) via TCP
ngrok http 8080 --log=stdout Mostra logs detalhados no terminal
ngrok http 8080 --basic-auth="user:senha" Protege o acesso com usuário e senha
ngrok http 8080 --host-header=rewrite Reescreve o header Host (útil para alguns apps)
ngrok config add-authtoken TOKEN Autentica o ngrok com sua conta
ngrok version Exibe a versão instalada
ngrok update Atualiza o ngrok para a versão mais recente

Interface web local do ngrok (inspecionar requisições):

navegador 
http://127.0.0.1:4040
07
Alternativa: Cloudflare Tunnel (URL fixa)

Se tiver um domínio próprio no Cloudflare, use o tunnel para ter uma URL permanente:

cmd / powershell 
# Instalar cloudflared
winget install --id Cloudflare.cloudflared

# Autenticar
cloudflared tunnel login

# Criar túnel
cloudflared tunnel create monitor-mikrotik

# Expor porta 8080
cloudflared tunnel --url http://localhost:8080
Vantagem: URL permanente. Requisito: domínio próprio cadastrado no Cloudflare.
