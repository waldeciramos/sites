Como usar
1
Execute o script no Debian remoto
Conecte-se ao servidor Debian e rode bash /root/tunnel-start.sh. O script inicia o túnel e disponibiliza as informações de acesso na porta 9999.
2
Obtenha a URL pública do painel
Após rodar o script, rode no Debian: nohup cloudflared tunnel --url http://localhost:9999 > /tmp/cf-painel.log 2>&1 & sleep 10 && grep -o 'https://[a-z0-9-]*\.trycloudflare\.com' /tmp/cf-painel.log | tail -1. Cole a URL gerada no campo abaixo.
3
Copie e use no MobaXterm ou CMD
O painel exibirá o Remote Host, Username e o comando completo pronto para colar no MobaXterm ou no terminal.
Conectar ao servidor
ℹ Rede local: informe o IP do Debian (ex: 192.168.70.10). Rede externa: informe a URL pública do segundo túnel (ex: xxx.trycloudflare.com). O script tunnel-start.sh deve estar em execução.
